# SEO Action Plan: Moh-AI Tech

This action plan was generated by the **Agentic SEO Skill** based on a technical audit of the current Next.js codebase.

## 🔴 High Priority (Immediate Impact)

### 1. Missing `robots.txt`
**Finding:** No `robots.txt` exists in the `public/` directory or as a dynamic `src/app/robots.ts` route.
**Impact:** Search engines and AI crawlers do not have explicit crawl policies or a pointer to your XML sitemap, which can delay indexing of new pages.
**Fix:** Create `src/app/robots.ts` to export standard crawler directives and point directly to `https://moh-ai-tech.com/sitemap.xml`.

### 2. Lack of Organization / LocalBusiness Schema
**Finding:** The homepage (`src/app/page.tsx`) does not contain `application/ld+json` structured data.
**Impact:** Answer engines (like ChatGPT, Perplexity) and Google Knowledge Panels struggle to confidently associate "Moh-AI Tech" with its founder, location (Namakkal, India), and core enterprise offerings.
**Fix:** Inject JSON-LD Schema on the homepage defining the `Organization` or `LocalBusiness`, including the MSME registration number (`UDYAM-TN-14-0088280`), founder details, and official social/GitHub links.

## 🟡 Medium Priority (Performance & AEO)

### 3. Image Optimization (`next/image`)
**Finding:** Some images and media elements may be using standard `<img>` tags or missing explicit lazy loading/priority hints.
**Impact:** Negative impact on Core Web Vitals (Largest Contentful Paint) which is a direct ranking factor for Google.
**Fix:** Ensure all heavy above-the-fold assets use Next.js `<Image priority />` and that below-the-fold assets are properly lazy-loaded.

### 4. Answer Engine Optimization (`llms.txt`)
**Finding:** The site lacks an `llms.txt` file.
**Impact:** AI agents and enterprise chatbots crawling the web for B2B vendors often struggle to parse rich CSS/HTML layouts.
**Fix:** Add an `llms.txt` at the root directory containing a clean, markdown-formatted summary of Moh-AI Tech's capabilities, target audience, and contact protocols specifically formatted for LLM ingestion.

## 🔵 Low Priority (Internationalization)

### 5. Hreflang Configuration
**Finding:** The site uses a `<LanguageProvider>` indicating multilingual support (English/Tamil), but `hreflang` tags are missing from the SEO `metadata` block in `layout.tsx`.
**Impact:** Search engines might not serve the correct localized version to users, or could misinterpret translated pages as duplicate content.
**Fix:** Add `alternates: { languages: { 'en-US': '/en', 'ta-IN': '/ta' } }` (or similar, depending on your routing setup) to the Next.js Metadata object in `layout.tsx`.
